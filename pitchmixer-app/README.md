# PitchMixer App

This is the frontend application for **VoiceMixer**.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

For more detailed information about the project, please refer to the [root README](../README.md).
